# rod-proposal

A modern full stack Digital Management Suite for restaurants, featuring:
- React (Vite) frontend (client/)
- Node.js (Express) backend (server/)
- PostgreSQL/Supabase integration
- Supabase Auth for authentication
- Accessibility (A11y) best practices
- CSS Grid/Flexbox only (no Bootstrap)

## Getting Started

1. Clone this repo
2. Install dependencies in both `client` and `server`
3. Set up environment variables (see `.env.example` in each folder)
4. Run the backend and frontend

## Folder Structure
- `client/` - Frontend (React)
- `server/` - Backend (Node.js/Express)
- `.github/` - Project instructions

## Accessibility
All UI components use `aria-label` and `alt` tags for A11y compliance.
